# Dashboard Assets

Drop the following PNG files in this folder to replace the text fallbacks shown in the header.

| File | Purpose | Recommended Size |
|---|---|---|
| `client-logo.png` | Mount Edgecombe Country Club Estate crest (boar / tusker on banner with wordmark) | 256 x 256 px, transparent or dark navy bg |
| `contractor-logo.png` | DF Strange Builders mark | 240 x 96 px landscape, transparent or white bg |

If a file is missing the header gracefully falls back to a text monogram.
